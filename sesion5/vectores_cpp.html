#Introduccion

Un evstor o array unidimensional es un tipo de dato que almacena una secuencia de datos del mismo TIPO ( en c++ un vector/arreglo solo puede almacenar algo de su mismo tipo, es un array de strings, ints, floats... en js esto es dinamico aqui no.). LOS ELEMENTOS de un vector se almacenan en zonas contiguas ( adyacentes, osea se almacenan de forma seguida ) de memoria y se puede acceder a ellos de manera directa mediante un INDICE o posicion 


            //Forma de declarar un arregllo desde iniciializacion
            int numero[] = { 1, 4, 5, 7, 8 };

            //forma de declarar un array SIN INICIALIZACION en c++
            int numero[10] // un vector de 10 posiciones ( solo le caben 10 posiciones )
